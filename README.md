## 実行方法
1. **本リポジトリのクローン**

```bash
git clone -b complete git@github.com:unchain-tech/Solana-NFT-Drop.git
```

2. **パッケージのインストール**

```bash
yarn install
```

3. **Candy Machine の構築**

4. **開発サーバーの起動**

```bash
yarn dev
```

ターミナル上に表示された URL にアクセスしましょう。
